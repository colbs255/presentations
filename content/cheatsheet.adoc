=== Platform

- stacks are stored in off-heap memory
- stacks take ~1Mb of memory (configurable)
- >2Kb metadata
- Switching: 1-10 microseconds

=== Virtual

- 200-300Bytes of metadata
- pay-as-you-go : on heap
- 200 nanoseconds
